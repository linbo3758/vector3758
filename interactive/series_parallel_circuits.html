Electricity: Series and Parallel Circuits
Diagram A
Series Circuit Diagram
Current (I) SAME everywhere
Voltage (V) SPLIT across components
Key Concept: Since there is only 1 loop, the rate of flow (Current) cannot change. However, the energy per charge (Voltage) must be shared between the bulbs: Vtotal = V1 + V2.
Diagram B
Parallel Circuit Diagram
Current (I) SPLIT down branches
Voltage (V) SAME on each branch
Key Concept: The junctions split the traffic (Current): Itotal = I1 + I2. But because each charge only travels through one single branch, it delivers its full energy payload to that branch's bulb: Vtotal = V1 = V2.
